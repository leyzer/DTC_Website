Required columns:

  • player_one - Player 1 username or full name

  • p1_faction - Player 1's faction/army name
  • p2_faction - Player 2's faction/army name
  • result - "Player 1 Wins", "Player 2 Wins", or "Drawn"
  • location - Store/location name
  • points_band - Game point value (e.g., "1000", "1500", "2000", "Combat Patrol")
