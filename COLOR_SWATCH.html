Brand Usage Guidelines

  • Primary: Navy (#1a3a52) - Use for main headers, navigation, primary buttons, and dominant design elements
  • Accent: Gold (#c9a961) - Use for highlights, borders, badges, and secondary buttons
  • Background: Cream (#f5e6d3) - Use for light backgrounds, cards, and contrast against navy
  • Text: Navy on white/cream for main content, white text on navy for emphasis
  • Hover States: Use Light Navy (#2d5a8c) for interactive elements
